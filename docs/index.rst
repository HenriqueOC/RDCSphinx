Bem-vindo a documentação do RDC-ARQ!
===================================
Visão geral

A elaboração de estudos que indiquem as soluções viáveis de implantação de um repositório arquivístico digital confiável - RDC - Arq para o TJDFT

Objetivo Geral

Apresentar alternativas de soluções para implantação de um repositório arquivístico digital confiável - RDC - Arq, integrado com solução de gerenciamento documental, tornando possível a elaboração e disseminação de modelo tecnológico de mapeamento, preservação e disseminação das informações associadas.
